# 🚁 Sharda University Drone Security & Surveillance System

This project is an AI-based autonomous drone security system designed for monitoring the campus of Sharda University.

The system uses computer vision and real-time monitoring to patrol university classrooms, corridors, and campus areas. The drone can:

* 🔍 Monitor classrooms and campus zones
* 🛡️ Detect suspicious activities
* 📹 Capture live video feed
* 🤖 Use AI-based object detection
* 🌐 Provide real-time alerts to security

The project is built using:

* Python
* OpenCV (cv2)
* AI/ML concepts
* Web-based dashboard (HTML, JS)

This system improves campus security, reduces manual monitoring, and provides smart surveillance using autonomous flying drones.
